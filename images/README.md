### Recommended Images

#### Service Photos
- Car audio installations
- CCTV systems
- Security installations
- DSTV dishes
- Before/after photos

#### Business Assets
- Company logo (if available)
- Team photos (optional)
- Vehicle photos (mobile service fleet)
- Workshop/office photos

#### Hero Section
- High-quality hero image showcasing work
- Or use the animated speaker component (already included)

## Image Guidelines

- **Format**: JPEG for photos, PNG for graphics with transparency
- **Size**: Optimize for web (compress images)
- **Naming**: Use descriptive names (e.g., `car-audio-install-1.jpg`)
- **Alt Text**: Add descriptive alt text in HTML for accessibility
- **Dimensions**:
  - Hero images: 1920x1080px or similar widescreen
  - Service cards: 800x600px
  - Thumbnails: 400x300px

## Optimization Tools

- [TinyPNG](https://tinypng.com/) - Compress PNG and JPEG
- [Squoosh](https://squoosh.app/) - Image optimization
- [ImageOptim](https://imageoptim.com/) - Mac app for compression
